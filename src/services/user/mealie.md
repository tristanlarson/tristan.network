Mealie is great for all the normal functions, but the best one imo is the import function that imports recipes from online recipe sites you find. If you find a random recipe on the internet, you can put the url in mealie and it'll import it to your database, credit given at the bottom.

You should definitely set this up if you're tired of cookbooks. Also, it's useful if you're in a shared living environment where you buy groceries together because it has a built in shopping list feature.
