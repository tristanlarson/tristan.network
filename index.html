Navidrome

Navidrome is a nifty music server that you can use to stream your old iTunes m4a files for sentimentality. Or, if you like collecting your favorite music in hi-fi quality like me, you can use it to server flac files from Qobuz. Either way, there's a certain romance that comes with Navidrome and that's what I like about it.

There are many clients for Navidrome, including one really great iOS app called Arpeggi. It is still in beta, so you'll need to install it via testflight.

One aspect of Navidrome I struggled with was the way it handles metadata. Having previously used Plex, it was weird that some of the metadata and artwork didn't populate correctly, and at first I just went right back to Plex. I gave it another shot one day, and after editing my music files' metadata with mp3tag, I was able to clean up the navigation so there weren't a hundred albums under "various artists" lol. Along the way I learned there are a few tricks to how Navidrome handles album and artist artwork differently from Plex, I find it's best to name all the artwork folder.jpg haha. Anyways, here are some tips.

If this sounds like too much work to you I reccommend you just use Plex with the PlexAmp app (which is almost as beautiful as Arpeggi btw). There are a lot of features locked behind the subscription to the Plex Pass, but it's a super smooth experience.

Navidrome Tips

When I was setting up Navidrome, I had two main problems:
1. My metadata was a little out of order, so my artists folder was messed up.
2. The artwork I had wasn't being read properly like it had been in Plex.

Metadata

To fix the metadata, I used a tool called mp3tag. There are similar tools for linux, but mp3tag is the best one, so use that one lol. To begin with, I made a list of all the artists and albums within them that needed fixing. With mp3tag, while editing the relevant files, use Alt+T to show the extended tags. THIS IS WHAT TRIPPED ME UP FOR SO LONG LMAO. You wouldn't think they would matter, but sometimes they do, and Navidrome gets tripped up on these tags.

Artwork

To fix the artwork, the main problem I noticed was that albums with "disc" subfolders didn't have artwork. With albums like this, I put the artwork in each directory, including both discs and the root directory of the album. This works for me, and I don't feel like finding which specific folder it reads the artwork from.
Other than that, I found that if I had multiple albums in an artist and one of the albums didn't have artwork, it was usually because of the file name. For example, in The Beatles, if you have album.jpgs in each album except for With The Beatles, the artwork for With The Beatles might not show up. My fix for this was to just name all the artwork folder.extension lol.

Foundry Virtual Tabletop

Foundry is my favorite virtual tabletop, even if I wasn't considering the fact that you can self-host it. The alternative for me was roll20, and the difference between the two is like night and day. Not only are there way more cool features in Foundry, but there's also many open source mods you can use (and or buy) to enhance your gameplay. In my opinion, it is well worth it to buy and host Foundry if you're into that sort of thing lol.

NTFY

NTFY is incredibly useful because it lets you send notifications to your devices. Normally this wouldn't be all that special, but that's not the only cool part. It also lets you send push notifications to Apple devices :D. Along with the fact that it's relatively simple to set up, I really love this project.

One problem I had with NTFY is that for an embarrassing amount of time I couldn't find the correct AUR to install it. There are a couple different ones, ntfy, ntfy-git, ntfy-alertmanager, ntfysh, and ntfsh-bin. The correct one turned out to be ntfysh-bin lol.
