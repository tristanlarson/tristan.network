My Services

I have a bunch of services accessible to me, my friends and family. I'll go through some (not many tbh) of them here.

Cloudflared Tunnels

Cloudflared tunnels are the bomb. I used to forward ports like a chump, if you could even believe it.

Tailscale

Tailscale is your own personal VPN with all the hosts you have connected to it.

NTFY

NTFY is a useful server that you can use to send notifications from your server to your devices, including iOS devices :D.

Pi-hole

I know everyone knows what Pi-hole is and that uni students set it up a lot as projects, but here it is anyways lol. If you don't know what it is, definitely click on this one.

Gramps Web

I exported my families ancestry data into Gramps Web for privacy reasons, and it ended up being way better than whatever we were using lol.

Mealie

Mealie is a great platform for everything digital about cooking. Recipes, shopping lists, meal planners, it's got it all.

Foundry VTT

Foundry is a self-hostable virtual tabletop used by geeks to play DnD type games online.

Cloudflared Tunnel

Cloudflare tunnels are convenient ways of exposing HTTP services running on your machine to the public internet without forwarding ports. I use them to expose literally everything I have running except for game servers which use UDP and don't use HTTP (two requirements). They use Playit tunnels instead.

You can run it as a systemd service or in a docker container. Here's the official setup page.

Tailscale

I want to say Tailscale is a VPN server, but I know it's so much more than that, I just can't properly describe it. Basically, you can add hosts to your own VPN and connect to them from anywhere. It's like Plex for systems management.

It's super useful, open source, privately owned (like Steam :D), and based in Canada. It doesn't get much better than that lol.

Playit Tunnel

A Playit tunnel is very similar to a Cloudflared Tunnel, only it's used for game data, so it works with UDP and non-HTTP traffic.

NTFY

NTFY is incredibly useful because it lets you send notifications to your devices. Normally this wouldn't be all that special, but that's not the only cool part. It also lets you send push notifications to Apple devices :D. Along with the fact that it's relatively simple to set up, I really love this project.

One problem I had with NTFY is that for an embarrassing amount of time I couldn't find the correct AUR to install it. There are a couple different ones, ntfy, ntfy-git, ntfy-alertmanager, ntfysh, and ntfsh-bin. The correct one turned out to be ntfysh-bin lol.

Pi-hole

Pi-hole is a DNS caching service and proxy / DNS firewall thing. It get's all the DNS queries from the devices on your network, and allows / blocks / redirects them based on your settings.

For example, the default blocklist will block things like your Amazon Firestick sending telemetry to whatever random Amazon domain. It also blocks a lot of ads if you want, specifically the ones that are loaded from known ad domains.

Gramps Web

Gramps Web is a self hostable ancestry archiving server, like ancestry.com but hosted by you so you know where all your data is. I'm using this to build up my family tree because I know I'll probably care about this stuff one day lol.

Here's the link to the demo, the username and password are both "owner." If you're already using another ancestry site to do these things, you should know that it's not as hard as you'd think to transfer your data into Gramps Web.

Here's the official docker-compose.yml file.

Mealie

Mealie is great for all the normal functions, but the best one imo is the import function that imports recipes from online recipes you find.

You should definitely set this up, it takes 2 seconds if you're experienced with docker and it makes my life as the food_maker_roommate 1000% easier lol.

Foundry Virtual Tabletop

Foundry is my favorite virtual tabletop, even if I wasn't considering the fact that you can self-host it. The alternative for me was roll20.net, and the difference between the two is like night and day.

Not only are there way more cool features in Foundry, but there's also many open source mods you can use (and or buy) to enhance your gameplay. In my opinion, it is well worth it to buy and host Foundry if you're into that sort of thing lol.
